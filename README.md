# NFL-BMI-Analysis

##Overview
This project examines whether BMI varies systematically across NFL positions and what that reveals about the limitations of BMI as a descriptive measure of body composition in elite athletes.

## Research Question
Does BMI vary systematically across NFL positions, and how useful is BMI as a descriptive measure of body type in elite athletes?

## Data Source
Player data were collected from NFL team websites in March 2026 and limited to active roster players.

## Tools Used
- Excel
- PostgreSQL
- RStudio

## Project Files
- docs/
- scripts/
  `├──` R/    
  `└──` SQL/  
- data/

## Key Findings
- BMI varied more by position than by age group or team.
- Defensive players had slightly lower average BMI than offensive players.
- Position was the strongest predictor of BMI.

## Reproducibility
The analysis can be reproduced using the included scripts and data files.
